Introduction to Programming

Lesson 1.1: The Basics of the Web and HTML

Basic parts of the web

The web is an enormous network of computers that communicate with each other. The user accesses the web through a computer with a web browser. Through the internet the browser is able to obtain html documents from various servers using http. Those documents are then interpreted by the browser and displayed to the user in the manner which the programmers intended.

HTML

HTML is an acronym for HyperText Markup Language. It is the basis for everything on the web. HTML is made up of text content, which is "what you see" and markup, which is "how it looks."

HTML attributes - elements/tags

HTML documents consist of elements, which include various tags that tell the browser how to display the document.

Computers are stupid

Programmers like to say that computers are stupid because they do not think for themselves. They do exactly as they are told. So, any little typo or other minor error can cause a computer to fail to carry out the program. On the other hand, because they do exactly as commanded, this gives programmers a lot of power if they understand how to correctly provide the necessary commands.

Lesson 1.2: Creating a Structured Document

Developer tools (in the Browser)

The developer tools in Google Chrome allow one to see how the browser is interpreting the HTML and CSS code. This can be a helpful tool in making sure the website is displayed the way that it should be.

The tree-like structure of HTML

HTML code is structured in a tree-like manner, where each element can branch out and contain multiple other elements within it.

Programmer tools

Programmers use tools such as text editors which can help write code with fewer errors. They can automate some parts of the code-writing process, such as automatically generating a closing HTML tag when the programmer types in the opening tag.

Lesson 1.3: Adding CSS style to HTML structure.

CSS (Cascading Style Sheets)

CSS is used to give style to the HTML document. With CSS the programmer can easily change such things as the colors of the text and background, the font type and size, the margins, etc. By assigning classes to the various HTML elements, the programmer can then use CSS to simultaneously change all of the elements of the same class. This avoids repetion, which is more effecient and reduces the chances of error.
